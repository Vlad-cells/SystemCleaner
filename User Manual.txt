Короткий User Manual — System Cleaner

 1. Сканування системи

- Натисніть "Sprawdzenie stanu"  
- Програма покаже завантаження CPU, температуру, RAM, GPU

 2. Очищення та оптимізація

- Натисніть "Czyszczenie i optymalizacja"  
- Доступні функції:
  - Очистити тимчасові файли
  - Очистити кеш браузерів
  - Очистити системні логи
  - Очистити кошик
  - Завершити важкі процеси

 3. Видалення програм

- Натисніть "Odinstaluj programy"  
- Виберіть програму зі списку  
- Натисніть "Видалити вибране"

 4. Налаштування

- Натисніть "Ustawienia"  
- Можна змінити:
  - Тему інтерфейсу
  - Мову (UA, EN, PL, DE)
  - Увімкнути/вимкнути автозапуск

 Примітка

- Програма не потребує встановлення  
- Можна запустити через main.exe або main.py
