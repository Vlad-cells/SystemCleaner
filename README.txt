 System Cleaner

System Cleaner — це утиліта для оптимізації системи Windows.  
Програма дозволяє сканувати стан системи, очищати тимчасові файли, завершувати важкі процеси, видаляти програми та змінювати мову інтерфейсу.

 Вимоги

- Windows 10/11
- Python 3.10+
- Пакети: `psutil`, `GPUtil`, `ttkbootstrap`, `pystray`, `Pillow`, `cpuinfo`, `wmi`

Інструкція запуску

1. Встановити Python 3.10+
2. Встановити залежності:

```bash
pip install -r requirements.txt
